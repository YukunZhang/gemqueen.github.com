#####Latex table command
```latex
\begin{table}[h]

\small %control the table font size

\begin{tabular}

\end{tabular}

\end{table}

```
#####The order of font from smallest to largest
```latex
\tiny

\scriptsize

\footnotesize

\small

\normalsize

\large

\Large

\LARGE

\huge

\Huge

```
